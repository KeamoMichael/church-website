# Harmony Church Website

A modern, responsive website for Harmony Church featuring multiple pages including About Us, Sermons, Ministries, Events, and more.

## Features

- Responsive design for all devices
- Multiple pages: Home, About, Sermons, Ministries, Events, Groups, Missions, Serve, Visit, and New Here
- Modern UI with custom fonts (Manrope)
- Image assets and logo integration
- Clean, organized file structure

## File Structure

```
church-website/
├── index.html          # Home page
├── about.html          # About Us page
├── sermons.html        # Sermons page
├── ministries.html     # Ministries page
├── events.html         # Events page
├── groups.html         # Groups page
├── missions.html       # Missions page
├── serve.html          # Serve page
├── visit.html          # Visit Us page
├── new-here.html       # New Here page
├── style.css           # Main stylesheet
├── fonts/              # Custom fonts (Manrope)
├── assets/             # Images and logos
└── Design Reference/   # Design inspiration files
```

## Setup

1. Clone this repository
2. Open `index.html` in your web browser
3. No build process required - it's a static website

## Technologies Used

- HTML5
- CSS3
- Custom fonts (Manrope)
